# LearnFlow Documentation

LearnFlow is an **AI-powered Python tutoring platform** built entirely using AI Skills. Instead of writing application code manually, developers create Skills that teach AI agents (Claude Code and Goose) how to build, deploy, and manage the platform.

## Core Principle

> The Skills are the product. ALL application code is generated by AI agents using Skills.

## What is LearnFlow?

LearnFlow helps students learn Python through conversational AI agents:

| Role | Features |
|------|----------|
| **Student** | Chat with AI tutor, write & run code, take quizzes, view progress |
| **Teacher** | View class progress, receive struggle alerts, generate exercises |

## Architecture at a Glance

```
Student → Next.js Frontend → Triage Agent → Kafka Pub/Sub → Specialist Agents
                                                                    ↓
                                                            PostgreSQL (Dapr State)
```

**6 AI Agents**: Triage, Concepts, Code Review, Debug, Exercise, Progress

## Skills Built

| Skill | Purpose |
|-------|---------|
| `agents-md-gen` | Generate AGENTS.md for repositories |
| `k8s-foundation` | K8s health check + Helm install |
| `kafka-k8s-setup` | Deploy Kafka on K8s |
| `postgres-k8s-setup` | Deploy PostgreSQL on K8s |
| `fastapi-dapr-agent` | Deploy 6 AI microservices with Dapr |
| `nextjs-k8s-deploy` | Deploy Next.js frontend with Monaco |
| `mcp-code-execution` | MCP pattern for real-time AI context |
| `docusaurus-deploy` | Deploy this documentation site |
