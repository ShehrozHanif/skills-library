# AGENTS.md Format Reference

## Required Sections

### 1. Project Overview
Brief description of what the project does, its purpose, and key goals.
```
# Project Name
One-paragraph description of the project.
```

### 2. Directory Structure
Tree view of important directories and their purposes.
```
## Directory Structure
project-root/
├── src/          # Source code
├── tests/        # Test files
├── docs/         # Documentation
└── scripts/      # Utility scripts
```

### 3. Tech Stack
Languages, frameworks, and tools used.
```
## Tech Stack
| Category | Technology |
|----------|-----------|
| Language | Python 3.x |
| Framework | FastAPI |
```

### 4. Conventions
Coding standards, naming patterns, and project-specific rules.
```
## Conventions
- File naming: snake_case for Python, kebab-case for configs
- Commit messages: "Agent: description"
```

### 5. Getting Started
How to set up and run the project.
```
## Getting Started
1. Clone the repo
2. Install dependencies
3. Run the project
```
